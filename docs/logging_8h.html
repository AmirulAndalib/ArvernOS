Detailed Description

This is the general kernel logging module.

It supports several logging levels such as INFO, DEBUG, and TRACE:

  • DEBUG logs are not enabled by default. They can be enabled by setting ENABLE_KERNEL_DEBUG=0 in config.
  • TRACE logs are not enabled by default. They can be enabled by setting ENABLE_KERNEL_TRACE=1 in config.

Each kernel module should ideally have its own logging header that depends on this one. The main use case is to have more configurable loggers that can be enabled/disabled depending on the needs.
